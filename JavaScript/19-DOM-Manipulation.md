# DOM Manipulation

## Selecting

`querySelector`, `querySelectorAll`, `getElementById`, `getElementsByClassName`.

## Creating and mutating

`document.createElement`, `textContent`, `innerHTML` (XSS risk with untrusted strings), `append`, `classList`.

## Attributes and data

`setAttribute`, `dataset` for `data-*` attributes.

## Modern APIs

`element.closest`, `matches`, `replaceChildren`.
