# Control Flow

## Conditionals

`if / else if / else`, ternary `cond ? a : b`.

## `switch`

Uses **strict equality** internally; remember **`break`** or `return`.

## Loops

`for`, `for...of` (iterables), `for...in` (object keys—usually avoid for arrays), `while`, `do...while`.

## `break` / `continue`

Labeled forms for nested loops.
