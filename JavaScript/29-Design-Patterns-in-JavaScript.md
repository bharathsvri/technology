# Design Patterns in JavaScript

## Module pattern

IIFE or ES modules exporting private state via closures.

## Factory / builder

Functions returning configured objects.

## Observer / pub-sub

`EventTarget`, Node `EventEmitter`, or custom channels.

## Singleton (caution)

One shared instance—prefer explicit DI in apps.

## Strategy / policy

Pass comparator functions to `sort`, validation strategies, etc.
