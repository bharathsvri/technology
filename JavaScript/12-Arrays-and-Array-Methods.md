# Arrays and Array Methods

## Mutating vs non-mutating

- **Mutating**: `push`, `pop`, `splice`, `sort`, `reverse`.
- **Non-mutating copies**: `slice`, `concat`, spread `[...arr]`, `toSorted` / `toReversed` (modern).

## Functional transforms

`map`, `filter`, `reduce`, `flatMap`, `some`, `every`, `find`, `findIndex`.

## Iteration

`for...of`, `forEach` (no `break`—use `for` or `some`).

## Typed arrays

`Uint8Array` etc. for binary data and WebGL/streams.
