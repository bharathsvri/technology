# Introduction to React

React builds a **virtual representation** of the UI and reconciles it to the DOM (or native views via React Native). **Unidirectional data flow**: props down, events/callbacks up.

## Why React?

Large ecosystem, **hooks** for reuse, **SSR** frameworks (Next.js), strong TypeScript support.

## Create apps

`npm create vite@latest` with React template, or `create-react-app` (legacy) / Next.js for production SSR.
