# Testing with React Testing Library

## Philosophy

Test behavior users see—**not** implementation details of internal state.

## `render` + queries

`screen.getByRole`, `findByText`, `userEvent` for realistic interactions.

## `waitFor`

Async assertions until conditions met.

## MSW

Mock network at HTTP layer for integration-style tests.
