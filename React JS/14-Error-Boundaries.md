# Error Boundaries

## Class-only (today)

`static getDerivedStateFromError`, `componentDidCatch` to catch **render** errors in subtree.

## React 19+

Error boundary improvements may evolve—check release notes.

## What boundaries do not catch

Event handler errors, async errors outside React, SSR errors (framework-specific).
