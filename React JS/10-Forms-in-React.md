# Forms in React

## Controlled inputs

`value` + `onChange` — React is source of truth.

## Uncontrolled

`ref` + `defaultValue` for simple cases.

## Libraries

**React Hook Form**, **Formik** for validation, arrays, async submit.

## Accessibility

Label `htmlFor`, error announcements with `aria-*`.
