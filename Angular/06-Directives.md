# Directives

## Structural

- **`*ngIf`** — conditional DOM.
- **`*ngFor`** — list rendering; use `trackBy` for performance.
- **`*ngSwitch`** — multi-branch.

## Attribute directives

`ngClass`, `ngStyle`, custom directives with `@Directive`.

## Host bindings

`@HostBinding`, `@HostListener` for encapsulating DOM behavior on host element.
