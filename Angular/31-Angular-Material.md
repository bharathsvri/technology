# Angular Material

## What it is

Official **Material Design** component library: buttons, inputs, datepicker, table, dialog, snackbar, stepper, etc.

## Setup

`ng add @angular/material` — theme, typography, animations module.

## Theming

Sass-based **palettes** (`primary`, `accent`, `warn`) and **density** for compact layouts.

## Accessibility

Many components ship **ARIA** patterns and keyboard support—still validate with axe and manual testing.

## CDK relationship

Material builds on **Component Dev Kit (CDK)** primitives (overlay, a11y, collections).

## Bundle impact

Import only needed modules (`MatButtonModule`, …) or use **lazy-loaded** feature modules for large apps.
