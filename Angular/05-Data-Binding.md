# Data Binding

## Interpolation

`{{ expression }}`

## Property binding

`[prop]="value"` — DOM property or `@Input`.

## Event binding

`(click)="save()"` — host listener.

## Two-way binding

`[(ngModel)]` with `FormsModule` for template-driven forms; or **signals** + explicit `(input)` patterns.

## Attribute binding

`[attr.aria-label]="label"`
