# Angular DevTools

## Browser extension

Inspect **component tree**, **injector hierarchy**, and **change detection** cycles.

## Profiler

Record interaction → see which components ran change detection and how long—find hot spots when not using **OnPush**.

## Router tab

Visualize route tree and active routes (extension features vary by version).

## NgRx integration

Some versions integrate with store DevTools separately (Redux DevTools extension).

## Usage tips

Profile **realistic** user flows; micro-benchmarks in dev tools can mislead due to dev-mode double work.
