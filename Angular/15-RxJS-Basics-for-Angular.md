# RxJS Basics for Angular

## Observable

Lazy stream; `HttpClient` returns cold observables.

## Operators

`map`, `filter`, `switchMap`, `catchError`, `debounceTime`, `takeUntil` for unsubscribe pattern.

## `async` pipe

Subscribes in template and unsubscribes on destroy—reduces memory leaks.

## `toSignal`

Bridge RxJS → **Angular signals** for interop.
