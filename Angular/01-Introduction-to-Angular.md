# Introduction to Angular

Angular is a **platform and framework** for building client applications in **HTML and TypeScript**. It provides:

- **Components** with templates
- **Dependency injection**
- **Router**, **forms**, **HTTP client**
- **CLI** for scaffolding, build, and tests

## Versions

Angular uses **semantic versioning** (e.g. 17, 18, 19…). Check [angular.dev](https://angular.dev) for the current major version and migration guides.

## First app

`ng new my-app` → `ng serve` — creates standalone-based app by default in recent versions.
