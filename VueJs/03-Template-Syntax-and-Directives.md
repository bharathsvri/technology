# Template Syntax and Directives

## Interpolation

`{{ msg }}`

## Directives

- `v-bind` / `:` — attributes
- `v-on` / `@` — events
- `v-if`, `v-else-if`, `v-else` — conditional
- `v-for` — lists (`:key` required)
- `v-show` — CSS display toggle
- `v-model` — two-way binding on inputs/components

## Modifiers

`@click.stop`, `@submit.prevent`, `v-model.lazy`, `.number`, `.trim`.
