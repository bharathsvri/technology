# Introduction to Vue

Vue emphasizes **incremental adoption**—drop onto a page for widgets or build full SPAs with router and state management.

## Core ideas

- **Reactivity** — UI updates when data changes.
- **SFC** — `.vue` files combine template, script, style.
- **Composition API** — colocate related logic in `setup` / `<script setup>`.

## Official ecosystem

**Vue Router**, **Pinia**, **Vite** (recommended build tool).
