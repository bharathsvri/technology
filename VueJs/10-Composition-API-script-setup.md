# Composition API with `<script setup>`

## Macros

`defineProps`, `defineEmits`, `defineExpose`, `defineOptions` (with plugin/version).

## Top-level bindings

Auto-exposed to template—no `return { }` from `setup`.

## Imports

Components auto-registered when imported in script setup (Vue 3.3+ improved ergonomics).
