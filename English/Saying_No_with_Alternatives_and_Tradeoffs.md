# Saying No with Alternatives and Trade-offs

**No** without context feels like obstruction; **no + options** moves work forward: “**Not this sprint**, but we can **ship a flag-guarded** subset **or** push **analytics** to **next** release.”

---

## Structure

1. **Acknowledge** the goal.
2. **Constraint** (capacity, risk, compliance).
3. **Two** viable paths with **costs**.
4. **Recommendation**.

---

## Escalation

If **priorities** conflict, **surface** to the **decider** with written **trade-off**—avoid silent **queue jumping**.
