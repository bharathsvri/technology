# Spring Framework — Concepts Index

This folder is a **deeper conceptual guide** to Spring Framework (IoC, MVC, WebFlux, JDBC/JPA, security, transactions). Each numbered topic is written for **learning and revision**: tables, pitfalls, and “when to use” guidance—not only one-liners. For Spring Boot–specific production topics, see **`Spring Boot Concepts/`**.
